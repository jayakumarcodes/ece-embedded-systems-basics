# ECE Embedded Systems Basics

## Description
Embedded C practice programs and notes for ECE (Electronics and Communication Engineering) placements.

## Topics Covered
- **Microcontroller Simulation**: GPIO control, digital I/O operations
- **Timer Interrupts**: Interrupt Service Routines (ISR), timer configurations
- **UART/Serial Communication**: Data transmission and reception
- **Bitwise Operations**: Bit manipulation for hardware control
- **Memory Management**: Stack, heap, and embedded constraints

## Files
### C Programs
- `gpio_blink_simulation.c` - LED blinking simulation using delays
- `timer_interrupt_example.c` - Timer ISR implementation
- `uart_communication.c` - Serial UART example
- `bitwise_operations.c` - Bit manipulation patterns

### Documentation
- `EMBEDDED_BASICS.md` - Theory and concepts
- `SETUP_GUIDE.md` - How to compile and run programs

## How to Use
1. Clone or fork this repository
2. Read `SETUP_GUIDE.md` for compilation instructions
3. Review the C files and theory in `EMBEDDED_BASICS.md`
4. Compile using: `gcc -o program program.c`
5. Run: `./program`

## Learning Outcomes
- Understand embedded C syntax and concepts
- Practice low-level hardware interaction
- Prepare for ECE placement interviews
- Build a foundation for IoT and VLSI projects
